HTML Practice (Groups)

Test

  • Create basic HTML page with one student’s GitHub Pages page.
  • The page must include these HTML tags:
    • headings (h1, h2, etc.), paragraph, div, span, strong, em, table, lists (ol and ul), and image
  • Do this in iterations, where each student must contribute something that includes:
    • Comment with purpose, source or inspiration for added code, and new code.
    • Edited code should also be commented on to detail changes.
    • Then the group commits the changes and pushes them to GitHub with proper commit comments.
  • Write in Reflection Journal.

Javascript (Groups)

Create a page that does the following:

  • Create basic HTML/CSS + JS page with first student’s GitHub Pages page (rotation to the first student again)
  • Include:
    • Declare a variable and set its value.
    • Use an array (AKA a table of variables stored as a set).
    • Create a function.
    • Throw in some conditional logic (AKA an if-then statement or a switch of some sort).
    • Create a loop that runs multiple times until some end condition is met.
    • Make some element execute the function when clicked on or affected by the user.
  • Refleciton Journal
